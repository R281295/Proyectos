Proyecto de finalización de curso de la academia AEPI (Asociación Española de Programadores Informáticos)

Consiste en realizar las funciones básicas de una BBDD (CRUD).

Cuenta con una interfaz gráfica (SWING) 

Inicialmente, el proyecto usaba MySQL y JDBC para realizar consultas y ejecutar procedimientos almacenados

Desde entonces se han echo diversos cambios, como limpieza y reestructuración de código,
y actualmente se está usando Maven (antes era JavaSE), Hibernate y SQLite como BBDD.
